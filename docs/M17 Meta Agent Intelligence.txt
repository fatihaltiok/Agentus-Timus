 M17: Meta-Agent Intelligence Upgrade — Implementierungsplan

 Context

 Der Meta-Agent delegiert Aufgaben an Spezial-Agenten, erhält aber keine strukturierten Rückantworten — nur rohe Strings. Es gibt keinen automatischen Blackboard-Write nach Delegationen,
 kein Replan-Protokoll bei Misserfolg und keine Übersicht der verfügbaren Agenten-Fähigkeiten. Ziel ist ein bidirektionales Kommunikationsprotokoll: strukturierte AgentResult-Objekte,
 automatisches Blackboard-Schreiben, dynamische Capability-Map und Replan-Loop.

 @staticmethod
 def _auto_write_to_blackboard(agent_type: str, task: str, result: str, status: str) -> str:
     """Schreibt Delegationsergebnis automatisch ins Blackboard. Gibt Key zurück."""
     try:
         from memory.agent_blackboard import get_blackboard
         bb = get_blackboard()
         key = f"delegation:{agent_type}:{int(time.time())}"
         ttl = {"success": 120, "partial": 60, "error": 30}.get(status, 60)
         bb.write(
             key=key,
             value={"task": task[:200], "result": result[:1000], "status": status},
             agent_id="agent_registry",
             ttl_minutes=ttl,
         )
         return key
     except Exception as e:
         log.warning("Auto-Blackboard-Write fehlgeschlagen: %s", e)
         return ""

 Schritt 4 — Meta-Agent: Rich Context + Capability Map

 _get_blackboard_summary() — echte Delegation-Einträge zeigen

 def _get_blackboard_summary(self) -> str:
     try:
         from memory.agent_blackboard import get_blackboard
         bb = get_blackboard()
         entries = bb.search("delegation:", agent_id=None, limit=5)
         if not entries:
             return "Blackboard leer"
         lines = []
         for e in entries:
             val = e.get("value", {})
             lines.append(f"- [{val.get('status','?')}] {e['key']}: {val.get('task','')[:60]}...")
         return "\n".join(lines)
     except Exception:
         return "Blackboard nicht verfügbar"

 Neue Methode _get_capability_map() hinzufügen

 def _get_capability_map(self) -> str:
     try:
         from agent.agent_registry import AgentRegistry
         reg = AgentRegistry.get_instance()
         lines = []
         for spec in reg.list_agents():
             lines.append(f"- {spec.agent_type}: {', '.join(spec.capabilities[:3])}")
         return "\n".join(lines) if lines else "Keine Agenten registriert"
     except Exception:
         return "Capability-Map nicht verfügbar"

 Schritt 5 — REPLAN-PROTOKOLL in META_SYSTEM_PROMPT

 Datei: agent/prompts.py — vor dem abschließenden """ des META_SYSTEM_PROMPT einfügen

 REPLAN-PROTOKOLL:
 Wenn delegate_to_agent status="partial" oder status="error" zurückgibt:
 1. Analysiere den Fehler im Ergebnis
 2. Wähle einen anderen Agenten ODER formuliere die Aufgabe konkreter neu
 3. Maximal 2 Replan-Versuche pro Sub-Task
 4. Nach 2 Fehlversuchen: status="partial" zurückgeben mit Erklärung
 Niemals denselben fehlgeschlagenen Call ohne Änderung wiederholen.
