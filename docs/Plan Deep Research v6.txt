 Rollback-Strategie

 Jede Phase ist additiv — bestehender Code wird nicht entfernt, nur ergänzt.
 Feature-Flags ermöglichen sofortiges Deaktivieren:
 DEEP_RESEARCH_YOUTUBE_ENABLED=false   # Phase 2 deaktivieren
 DEEP_RESEARCH_IMAGES_ENABLED=false    # Phase 4 deaktivieren
 DEEP_RESEARCH_PDF_ENABLED=false       # Phase 5 deaktivieren
 Bei Problemen in Phase N → Flag setzen → System läuft auf v5.1-Stand.
