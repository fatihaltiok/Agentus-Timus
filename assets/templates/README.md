# Template Assets (OpenCV Fallback)

Dieses Verzeichnis enthält UI-Templates für den OpenCV-Fallback in Milestone 3.

Konventionen:
- Dateiformate: `.png`, `.jpg`, `.jpeg`, `.webp`, `.bmp`
- Benennung: sprechend und stabil, z. B. `login_button.png`, `send_icon.png`
- Quellen: möglichst verlustfrei ausgeschnittene Referenzbereiche

Hinweis:
- Das Tool `opencv_template_match` kann entweder
  - direkt per `template_path` arbeiten oder
  - per `template_name` in diesem Verzeichnis suchen.
