# Timus Branding Assets

Primary logo and icon assets for Timus.

## Files

- `assets/branding/timus-logo-primary.svg`
  - Transparent background
  - Full lockup (symbol + wordmark)
  - Recommended for README, docs, website hero

- `assets/branding/timus-icon.svg`
  - Transparent background
  - Symbol only
  - Recommended for app icon, avatar, favicon source

- `assets/branding/timus-logo-glow.png`
  - Glow variant on dark background
  - Recommended for README hero preview and social teaser images
  - Size: `1536x1024`

## Color Palette

- `#1e2a42` wordmark
- `#34b9ee` / `#0581c7` blue nodes
- `#53d87b` / `#1fae48` green nodes
- `#0f6cb6` dashed orbit stroke

## Export Tips

- Keep SVG as the source of truth.
- Export PNG sizes: `512`, `1024`, `2048`.
- Prepare monochrome variants for dark and light surfaces.
